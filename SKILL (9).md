# Raw Sources

Immutable. Drop source documents here — articles, transcripts, papers, exported notes, clipped web
pages. Claude reads from this folder but never edits or deletes anything in it; it's the source of
truth the wiki is built from. If a source turns out to be wrong or outdated, don't delete it — the
wiki pages built from it should note it's superseded, but the raw record stays.

Downloaded images from clipped sources go in `assets/`.
